# Installation Steps



## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) Disclaimer:** Built purely for demonstration of React/TailwindCSS/Redux/NextAuth/Firebase/Stripe API Payment skills; zero affiliation with the original brand."

4) ```yarn run dev```


### If you see the window below then you are set to build AMAZON 2.0!
